# RS485/RS232


The AIBOX-186 has an RS485 interface, with the device name `/dev/ttyS4`, supporting half-duplex, and a default baud rate of `9600`.

There is also an RS232 interface, with the device name `/dev/ttyS2`, supporting full-duplex, and a default baud rate of `115200`.

The testing method can be directly referenced in the Core-1688JD4 wiki page under the section "[UART Debugging Methods](https://wiki.t-firefly.com/en/Core-1688JD4/driver_uart.html#)".
