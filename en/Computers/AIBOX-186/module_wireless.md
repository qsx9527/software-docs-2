# Wireless Modules
The communication modules supported by Firefly are as follows:
- [WXT14R2201 WiFi/BT Module](https://wiki.t-firefly.com/RTL8852BE/wxt14r2201.html)
- [WCT1FR2201 WiFi/BT Module](https://wiki.t-firefly.com/RTL8822CE/wct1fr2201.html)
- [EC20 4G-LTE Module](https://wiki.t-firefly.com/EC20/ec20.html)
- RM500U-CN 5G Module
