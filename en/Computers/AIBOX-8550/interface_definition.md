# Interface Definition

AIBOX-8550 provides these interfaces:

* DC Power Port
* 2 x USB3.0
* 1 x Type-C (For firmware download)
* 1 x Console (For Debug)
* 2 x 1000M Ethernet RJ45
* 1 x HDMI 2.0
* 1 x TF Card slot
* Power Key
