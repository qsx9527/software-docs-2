# Interface definition

AIBOX-1684X has rich interfaces, mainly including:
- 12V Power interface (5.5*2.5mm)
- Power button
- 1000Mbps Ethernet x 2
- USB 3.0 x 2
- TF card slot
- Type C (Debug serial)
