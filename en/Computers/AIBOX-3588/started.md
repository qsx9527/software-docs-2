# Introduction
**AIBOX-3588** Equipped with Rockchip octa-core 64-bit processor RK3588, it adopts advanced process technology, high performance and low power consumption, built-in ARM Mali-G610 MP4 quad-core GPU, integrated 6 TOPS computing power NPU, supports private deployment of ultra-large-scale parameter models under Transformer architecture, supports traditional network architectures such as CNN, RNN, LSTM, supports multiple deep learning frameworks, supports custom operator development, and supports Docker container management technology. One box can meet personalized AI deployment.
