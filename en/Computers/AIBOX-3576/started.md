# Introduction
**AIBOX-3576** is equipped with the Rockchip octa-core 64-bit AIOT processor RK3576. It utilizes advanced manufacturing processes for high performance and low power consumption. The device features an ARM Mali G52 MC3 GPU and integrates a 6 TOPS computing power NPU, supporting private deployment of large-scale parameter models under the Transformer architecture. It also supports traditional network architectures such as CNN, RNN, and LSTM, various deep learning frameworks, and custom operator development. Additionally, it supports Docker container management technology, making it a single-box solution for personalized AI deployment.
