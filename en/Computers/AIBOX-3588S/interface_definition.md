# Interface definition

AIBOX-3588S has rich interfaces, mainly including:
- 12V Power interface（5.5*2.5mm）
- Power button
- MaskRom button
- 1000Mbps Ethernet x 1
- USB 3.0 x 2
- PCIE/SATA SSD hard disk card slot (bottom of the machine) x 1
- HDMI
- TF card slot
- Console (Debug serial)
- Type-C（Upgrade）
- Work indicator
