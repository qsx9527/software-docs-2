# Interface Definition

AIBOX-9075 provides these interfaces:

* DC Power Port
* 2 x USB3.0
* 1 x Type-C (For debug console)
* 2 x 2.5G Ethernet RJ45
* 1 x HDMI 2.0
* 1 x SIM Card Slot
* 2 x GMSL2 4Pin Mini FAKRA
* 2 x RS485
* 2 x CAN-FD
* 6 x IO in
* 6 x IO out
* Download Key
* Power Key
