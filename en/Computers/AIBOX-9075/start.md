# Start

## Power On

Connect the power supply. Wait a while, the indicator led will on, then wait it to boot.

After the startup is completed, the screen will light up and display the system desktop.

## System Info

* Version: Ubuntu 24.04
* Display Protocol: Wayland
* Desktop: Gnome
* User: firefly
* Password: firefly
* Root Password is not set
