# NPU use

AIO-3399C(AI) has a built-in GTI 2801S NPU module.

***Note: To use the NPU function, please purchase the AIO-3399C(AI) hardware version with SPR2801S chip attached***

## Instructions

For how to use the AI module, please refer to [NCC S1](http://wiki.t-firefly.com/NCCS1/) related documents.

**Note: The AI module device node is `/dev/sgX`, usually `/dev/sg0`. **
