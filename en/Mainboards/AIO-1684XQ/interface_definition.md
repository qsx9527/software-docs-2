# Interface definition

## Interface definition of the whole machine

AIO-1684XQ provides rich interfaces, including:

* 12V power interface (5.5*2.5mm)
* 4 x USB 3.0
* 2 x Gigabit Ethernet
* TF card slot
* SIM card slots x 3
* Fan interface
* Power button
* Reset button
* Debug serial port (labeled as "Debug" in the diagram)
* Industrial serial ports (RS485, TTL)
* Mini PCIe interface (4G LTE)
* M.2 interface (5G Module)
* SATA interface (SATA 3.0 HD)
