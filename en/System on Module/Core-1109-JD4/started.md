## Introduction

**Core-1109-JD4** uses Rockchip RV1109 series processors, with built-in AI neural network acceleration NPU computing power up to 1.2Tops, supports 5M H.265/H.264 multi-channel decoding, and supports H.265/H.264 multi-channel encoding of different specifications. Built-in 5 million ISP 2.0, with multi-level noise reduction, 3-frame HDR and other technologies, supports 3 cameras input at the same time, meets the industry needs of security products and AIoT, and is widely used in: face recognition, IPC smart network cameras, gate access control, smart security, smart finance/construction sites, smart travel and other industries.

![](../../../rv1126_img/Core-1126-JD4/Core-1126-JD4-front.png)  
      

The [AIO-1126JD4](https://item.taobao.com/item.htm?id=670519903322) development board consists of the core board Core-1109-JD4 + the base board MB-JD4-RV11091126. AIO-1126JD4 has rich interfaces such as RGMII, USB2.0, I2C, UART, GPIO, MIPI-DSI and MIPI-CSI, which can be directly applied to various smart products to accelerate product implementation. For details, please refer to [Interface Definition](interface_definition.md).
