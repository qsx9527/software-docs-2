# Hardware Information

## Hardware Resources

### CORE-1126-JD4

* <font color=blue>**CPU**</font>:**Quad-Core ARM Cortex-A7，RISC-V MCU**
* <font color=blue>**NPU**</font>:**2.0Tops, support INT8/ INT16**
* <font color=blue>**DDR**</font>:**DDR4 1GB/2GB**
* <font color=blue>**Flash**</font>:**eMMC 8GB/16GB, SD Card** 
* <font color=blue>**Display**</font>:**MIPI-DSI interface, 1080P@60fps**
* <font color=blue>**Camera**</font>:**Double MIPI-CSI interface**
* <font color=blue>**ISP**</font>:**14M ISP 2.0 with 3 frame HDR(Line-based/Frame-based/DCG)**
* <font color=blue>**Hardware Encoding**</font>:**Support 4K H.264/H.265 coding**</br>
   -3840 x 2160@30 fps+720p@30 fps encoding</br>
* <font color=blue>**Hardware Decoding**</font>:**Support 4K H.264/H.265 decoding**</br>
   -3840 x 2160@30 encoding + 3840 x 2160@30 fps decoding
* <font color=blue>**RTC**</font>:**External RTC**
* <font color=blue>**Network**</font>:**Gigabit Ethernet port, 100MB USB to Ethernet port, WiFi, 4G module interface**
* <font color=blue>**Sound Card**</font>:**RK809 integrated audio decoder**
* <font color=blue>**Peripheral Interface**</font>:**Debug serial port, USB interface, headphone interface, analog microphone interface, Speaker interface**

### CORE-1109-JD4

* <font color=blue>**CPU**</font>: **Dual-core ARM Cortex-A7，RISC-V MCU**
* <font color=blue>**NPU**</font>: **1.2Tops, support INT8/ INT16**
* <font color=blue>**DDR**</font>:**DDR3 1GB/2GB**
* <font color=blue>**Flash**</font>:**eMMC 8GB/16GB, SD Card** 
* <font color=blue>**Display**</font>: **MIPI-DSI interface，1080P@60fps**
* <font color=blue>**Camera**</font>:**Double MIPI-CSI interface**
* <font color=blue>**ISP**</font>: **5 million ISP 2.0 with 3 frame HDR (Line-based/Frame-based/DCG)**
* <font color=blue>**Hardware Encoding**</font>：**Support H.264/H.265 encoding**</br>
   -2688 x 1520@30 fps+1280 x 720@30 fps</br>
   -3072 x 1728@30 fps+1280 x 720@30 fps</br>
   -2688 x 1944@30 fps+1280 x 720@30fps</br>
* <font color=blue>**Hardware Decoding**</font>：**Support 5M H.264/H.265 decoding**</br>
* <font color=blue>**RTC**</font>:**External RTC**
* <font color=blue>**Network**</font>:**Gigabit Ethernet port, 100MB USB to Ethernet port, WiFi, 4G module interface**
* <font color=blue>**Sound Card**</font>:**RK809 integrated audio decoder**
* <font color=blue>**Peripheral Interface**</font>:**Debug serial port, USB interface, headphone interface, analog microphone interface, Speaker interface**
