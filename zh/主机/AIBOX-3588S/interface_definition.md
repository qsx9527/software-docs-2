# 接口介绍

AIBOX-3588S 接口丰富，主要包括：
- 12V 电源接口（5.5*2.5mm）
- Power 按键
- MaskRom 按键
- 千兆以太网 x 1
- USB 3.0 x 2
- PCIE/SATA SSD 硬盘卡槽（机器底部） x 1
- HDMI
- TF卡槽
- Console (调试串口)
- Type-C USB2.0（烧录）
- 电源指示灯
