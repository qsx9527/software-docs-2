# 简介
**AIBOX-3588S**  配置 Rockchip 八核64位处理器 RK3588S，采用先进工艺制程，高性能低功耗，内置ARM Mali-G610 MP4四核GPU，集成6 TOPS算力NPU，支持Transformer架构下超大规模参数模型的私有化部署，支持CNN、RNN、LSTM等传统网络架构，支持多种深度学习框架，并支持自定义算子开发，支持Docker容器化管理技术。一个盒子就能满足个性化的AI部署。
