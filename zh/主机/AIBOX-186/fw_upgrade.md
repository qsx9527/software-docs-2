# 固件升级

AIBOX-186 使用 TF 卡升级固件，用户需严格按照以下步骤操作。

## 准备工作

- AIBOX-186
- PC 机
- USB 读卡器
- TF 卡，建议 Class10 或更高规格, 并根据固件大小选择 8/16/32 GB 容量
- 待升级的固件

## 升级步骤

（1）TF 卡需要使用 MBR 分区，并格式化成 FAT32 格式；

（2）将待升级的固件（zip 压缩格式）里的全部文件解压到 TF 卡根目录上；

（3）TF 卡插入 AIBOX-186 的 TF 卡槽，然后上电开机；

（4）升级过程中，LED 灯会间歇短暂闪烁，表示升级正在进行；

（5）如果升级成功，绿色 LED 灯会一直闪烁，此时拔出 TF 卡，让机器重新下、上电；

（6）如果升级失败，所有 LED 灯都会灭掉。

注意事项：

- TF 卡如果超过 32GB，可能因 Windows 系统的限制而无法格式化成 FAT32 格式，故最好选择 32GB 以下大小的 TF 卡
- 升级时间有点长，大概要六分钟左右，请耐心等待
