# RS485/RS232

AIBOX-186 有一个 RS485 接口，设备名称为 `/dev/ttyS4`，支持半双工，默认波特率为 `9600`。

 还有一个 RS232 接口，设备名称为 `/dev/ttyS2`，支持全双工，默认波特率为 `115200`。


测试方法可直接参考 AIO-186JD4 维基页面的《[UART 调试方法](https://wiki.t-firefly.com/Core-186JD4/driver_uart.html#tiao-shi-fang-fa)》部分
