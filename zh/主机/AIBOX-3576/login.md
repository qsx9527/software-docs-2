# 登录


AIBOX-3576 登录方式有两种，一种是通过 Console 串口进行终端登录，一种是通过 HDMI 登录。

## Console 登录
Type-C 线接入 Console 口，登录账号为`root`，默认没有设置`root`密码。<br>
使用以下串口参数：
* 波特率：115200
* 数据位：8
* 停止位：1
* 奇偶校验：无
* 流控：无

## HDMI 登录
在界面登录的时候，自动登录`firefly`用户，`firefly` 用户密码也为`firefly`。
