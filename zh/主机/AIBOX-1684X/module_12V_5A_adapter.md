# 12V 电源适配器

## 产品参数

* 产品：电源适配器（桌面式）
* 规格：美规/欧规
* 输入标准：AC100-240V 50/60Hz
* 输出标准：12V-5A

**注意：** AIBOX-1684X 一体机正常工作需要电源 12V/5A，电流低于 5A 可能会因电流过小而异常重启，为了保证开发板的正常工作，请使用电压为 12V，电流为 5A  的电源，推荐使用 Firefly 官网电源配件。
