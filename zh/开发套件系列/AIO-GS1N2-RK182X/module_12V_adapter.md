
# [12V电源适配器](https://store.t-firefly.com/goods.php?id=31)

## 产品参数

* 产品：电源适配器
* 规格：美规/欧规
* 输入标准：AC110-240V 50/60Hz
* 输出标准：12V-2A

**注意：** RK182X开发套件 一体机正常工作需要电源 12V/2A，电流低于 2A 可能会因电流过小而异常重启，为了保证开发板的正常工作，请使用电压为 12V，电流为 2A~3A 的电源，推荐使用 Firefly 官网电源配件。
