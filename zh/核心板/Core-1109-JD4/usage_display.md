# Display 使用

* 底板与 MIPI 屏幕的硬件连接方式如图所示：

![](../../../rv1126_img/Core-1109-JD4/mipi_screen.jpg)

* **注：默认公版固件在底板不接摄像头的情况下只显示 firefly LOGO 图标。如果底板接了 0S04A10 摄像头，则屏幕显示摄像头预览界面。rv1126 的系统没有配备桌面环境。**

* 屏幕转向

```
#指定 QT 输出平台并使屏幕顺时针转向 90°
export QT_QPA_PLATFORM=linuxfb:rotation=90
```
* 如果屏幕出现花屏，或者不显示的情况。请检查LCD排线是否有问题，更换优质材质的短排线能解决花屏问题。
