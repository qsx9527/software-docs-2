# 接口定义

## 整机接口定义

AIO-1684XJD4 提供了丰富的接口，主要包括：

* 12V 电源接口（5.5*2.5mm）
* 2 x USB 3.0
* 2 x USB 2.0
* 2 x 千兆以太网
* WIFI 天线
* HDMI 1.4
* TF 卡槽
* SIM 卡槽
* 风扇接口
* Power 按键
* Reset 按键
* 调试串口（即图中 Debug 口）
* 工业级串口 (RS485, RS232, TTL)
* Mini PCIe 接口（4G LTE）
* M.2 接口（5G Module）
* M.2 SATA 接口（SATA3.0 SSD）
